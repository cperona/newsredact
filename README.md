# NewsRedact
A command line utility for News Redaction.
It lets the user add multiple editors, and each editor can create different news pages. The program lets the user select from a list of news types.
It also has a points and price system with different rules that determine them for each news page.

# Set Up
1. Clone the repository: `git clone https://github.com/cperona/newsredact.git`
2. Set the SDK to 'temurin-25'
